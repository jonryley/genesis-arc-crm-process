# Account Fields

## Core Fields

- account_name
- account_owner
- parent_account
- sector
- sub_sector
- geography
- account_tier
- strategic_account_flag
- current_customer_flag
- installed_base_summary
- annual_revenue_potential
- relationship_strength

- whitespace_opportunities

## Purpose

These fields describe the organisation level context and strategic importance of the account.
