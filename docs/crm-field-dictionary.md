# CRM Field Dictionary

This document defines the core fields used in Genesis Arc.

## Field Types

- Text
- Picklist
- Multi select
- Number
- Currency
- Date
- Boolean
- Long text
- Lookup

## Field Naming Style

Use consistent naming such as:

- account_name
- account_sector
- opportunity_stage
- customer_problem
- decision_maker_name
- value_hypothesis
- delivery_owner
- renewal_likelihood

## Reporting Categories

Each field should support at least one of these outcomes:

- Qualification quality
- Pipeline visibility
- Forecast confidence
- Conversion effectiveness
- Delivery readiness
- Growth potential
